HOW TO CHANGE THE LEFT HERO IMAGE:

1. Put your new image in this folder
2. Name it exactly: "current.jpg" 
3. Run deploy.bat
4. Your new image will appear on the website!

IMPORTANT:
- File must be named "current.jpg"
- Supported formats: .jpg, .jpeg, .png
- Recommended size: 300x200 pixels or similar ratio

The old image will be automatically replaced.
