## 🚀 Quick Start

### Deploy to Production
```bash
# Simple one-command deployment
deploy.bat
```

### Local Development
```bash
# Install dependencies
npm install

# Start local server
python -m http.server 8000
# Then visit: http://localhost:8000
```

## 🔧 Maintenance

### Adding New Location Pages
1. Copy existing location page (e.g., `gutter-guards-roswell-ga.html`)
2. Update city name, phone numbers, and local references
3. Add to `sitemap.xml`
4. Deploy with `deploy.bat`

### Updating Phone Numbers
All phone numbers are standardized to `(470) 851-6780`. If changes needed:
1. Search and replace across all files
2. Update schema markup in `index.html`
3. Test calculator functionality

### Content Updates
- **Homepage**: Edit `index.html`
- **Calculator**: Edit `calculator.html` 
- **Services**: Edit `services.html`
- **Deploy**: Run `deploy.bat`
